简单将我搭建Hexo博客的过程记录下来。

1. 前提条件

  • GitHub账号
  • 安装Node.js
  • 安装git

2. 安装Hexo

利用npm来安装Hexo。首先要知道Hexo有3.x和2.x的区别，Hexo官网给的应该是Hexo 3.x的安装教程，我就是按照官网的步骤来做，报错之后google来解决。

$ npm install -g hexo-cli

$ npm install hexo --save

安装完之后，创建博客文件夹

$ hexo init hysic-blog
$ cd hysic-blog
$ npm install

然后
